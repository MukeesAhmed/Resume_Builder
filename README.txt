This file contains on how to use the application

1) The application consists of 3 pages, first a main page from where you can start building your resume

2) Second page is the homepage where you will have the required fields to fill in your information

3) Once you are done with filling in the information, you will have 2 templates to select from. You can view either one of them

4) After that you can go with anyone of the desired template.
